If you want to compose a web application, you must consider following stuff:

  • The client side technology,
  • The server side technology,
  • The data persistent technology.

But if you want to do better, you should also think about:

  • Which is the fastest way to get through?
  • How to get a state-of-art UI?
  • Is there a way to fit all the screen resolution?
  • What is the best server technology which provides good performance yet low cost?
  • Which data persistent method do I need?
  • How to build a re-usable framework with good modularization?
  • How about the security problem?
  • How can I find all these technologies and compose them into a monolith?
